Getting Started

Buoy is a simple micro-library that makes working with vanilla JS a little bit easier.

  • Add, remove, toggle and check for classes.
  • Get an object's sibling elements.

How to use Buoy

1. Include Buoy on your site.

<script src="buoy.js"></script>

Make sure Buoy is included before any scripts that make use of it.

2. Use Buoy's functions in your scripts.

hasClass - Check if an element has a class.

var turkey = document.querySelector('#turkey');
if ( buoy.hasClass(turkey, 'sandwich') ) {
    // Do stuff...
}

addClass - Add a class to an element.

var turkey = document.querySelector('#turkey');
buoy.addClass(turkey, 'sandwich');

removeClass - Remove a class from an element.

var turkey = document.querySelector('#turkey');
buoy.removeClass(turkey, 'sandwich');

toggleClass - Add or remove a class from an element.

var turkey = document.querySelector('#turkey');
buoy.toggleClass(turkey, 'sandwich');

getSiblings - Get an array of siblings for an element.

var turkey = document.querySelector('#turkey');
var turkeyFriends = getSiblings(turkey);
[].forEach.call(turkeyFriends, function (friends) {
    // Do stuff...
});

New to vanilla JS? Check out Ditching jQuery for Vanilla JS.

3. Pat yourself on the back.

Seriously, you're done. Nice work! Modify as needed.

Browser Support

Buoy works in all modern browsers, and IE 6 and above.
